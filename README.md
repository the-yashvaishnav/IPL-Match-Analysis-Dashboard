# IPL-Match-Analysis-Dashboard

## 📌 Description
This project presents an interactive dashboard to analyze IPL match data, including team performance, match results, player statistics, and venue analysis. It provides insights into win/loss trends, top-performing players, and match outcomes, helping users understand key patterns and make data-driven decisions.

## 📊 Tools Used
- Tableau

## 📈 Key Insights
- Total Matches
- City Wise Match Count
- Top Players by Player of the Match Awards
- Toss Decision Distribution (Bat vs Field)
